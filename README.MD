# A simple Blog App

## Functionalities

- create a post
- see other posts
- delete a post

## Tech Stack

- Frontend - `ReactJS` + `MUI`
- Backend - `ElysiaJS` + `BunJS` + `Prisma`
- DB - `PostgreSQL`

- All Components are Dockerized

## Running it

- setup enviromental variables in `.env`

```bash
docker compose up -d
```
